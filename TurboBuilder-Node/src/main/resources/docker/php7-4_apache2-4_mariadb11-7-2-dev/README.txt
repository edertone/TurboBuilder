This is a docker project containing the following elements:

    PHP 7.4.33
    PHPUNIT 7.5.20
    APACHE 2.4
    MARIADB 11.7.2

The container folders are mapped to the following relevant folders in the project:

    <project>/target/logs/docker --> Will contain all the logs generated by the containers
    <project>/target/<projectname>/dist --> Will contain the executable files to be run by the container
    
Executable files can be run with a browser at:

    https://localhost/ or http://localhost/
    
Database is available at:

    localhost:3306 (or database:3306 from inside the container)
    user: root (No password required)
